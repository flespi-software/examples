# message converter

Script to convert flespi device messages to **JSON, KML, GPX, GEOJSON, CSV** formats

### Run script
```bash
python3 convert.py <token> <device_id> <from> <to> {JSON,KML,GPX,GEOJSON,CSV}
```

`from` and `to` - unix timestamp

`device_id` - flespi device id

`token` - flespi token
